# UtaiXMLについて
日本の伝統芸能である**能**のうち，声楽にあたる部分を**謡**という．<br>
**UtaiXML**は，謡の楽譜であり，また能の台本としての性質を持つ**謡本**のデジタル化の取り組みである．<br>
能の構造を示した理論的な背景に基づいて楽譜データに対する大部分の設計を行い，サンプルデータを作成した．

対象：「観世流昭和版」(著作権保護期間満了)より「羽衣」(1934)，小段「クセ」の冒頭「春霞」<br>
論文：「能の謡本に記された楽譜情報のデジタル化」(小泉 暢)，人文科学とコンピュータシンポジウム「じんもんこん2025」発表<br>
注意：**UtaiXML**は開発段階であるため，今後設計変更を行う予定あり
